# ZT_RKE2
Repository of supporting code for Honours project. 

# Building 

## Install Docker 

[Here](https://docs.docker.com/engine/install/ubuntu/)

## Build images

`docker-compose -f up app/docker-compose.yaml`


## Install K8S 
Run rke2_install.sh

## Install libpcap libraries

`sudo apt-get install libpcap-dev`
